# How to run it:

## Windows:
1. Run the following commands on the terminal:
```bash
python -m venv .venv
.venv/Scripts/activate
```
2. Then with pip install the behave library
```bash
pip install behave
```

3. Finally, run the behave command to run all the acceptance tests.
```bash
behave
```

## MacOS - Linux:
1. Run the following commands on the terminal:
```bash
python3 -m venv .venv
source .venv/bin/activate
```
2. Then with pip install the behave library
```bash
pip install behave
```

3. Finally, run the behave command to run all the acceptance tests.
```bash
behave
```
